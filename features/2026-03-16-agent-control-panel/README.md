## Summary
Кнопочный интерфейс управления режимами агентов через Telegram inline keyboard. Команда /panel в личке Сократа → панель с toggle-кнопками для 5 агентов (active/sleep) + brainstorm toggle. Только авторизованный пользователь (sender_id). Реализация: OpenClaw skill в Сократе с callback handler.
