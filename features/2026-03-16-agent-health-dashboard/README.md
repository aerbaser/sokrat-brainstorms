## Summary
Standalone SPA для визуального мониторинга 5 агентов Империи. Bun server + vanilla HTML/CSS/JS. Тёмная терминал-тема. Polling каждые 10 сек. Один API endpoint мёржит данные из JSON-файлов (brainstorm #1) + host metrics + Docker stats. Доступ через Cloudflare tunnel + Bearer token auth.
