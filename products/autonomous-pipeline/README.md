## Summary

Полностью автономный event-driven pipeline от идеи до delivery в multi-agent системе Платона. 9 фаз (Ideate → Research → Brainstorm → Decompose → Develop → Test → Review → Deploy → Retro), GitHub Labels как state machine, supervised auto mode с 30-мин auto-continue. Параллельная разработка через Архимеда, трёхуровневое тестирование (unit + integration + E2E), auto-deploy на staging.
